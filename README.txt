Para este ejercicio contamos con 2 divs, uno abajo del otro, cada uno con un ancho fijo y un color de fondo.

Deberemos agregarles las posiciones y reglas de CSS necesarias para que:

* El primer <div> se ubique 20px a la derecha y 30px hacia abajo de su posición original.
* El segundo <div> se ubique, en relación al body, a 30px de la derecha y a 20px de abajo.
